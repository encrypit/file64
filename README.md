# file64

Convert Base64 to Blob/File and Blob/File to Base64.

## Quick Start

```ts
import 'file64';
```

## Installation

[NPM](https://www.npmjs.com/package/file64):

```sh
npm install file64
```

[Yarn](https://yarnpkg.com/package/file64):

```sh
yarn add file64
```

## Usage

```ts
import 'file64';
```
